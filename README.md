친구가 원하는 물건을 다른 사람들과 **펀딩을 통해 선물**하세요!

## 프로젝트 소개
```
KaKao API를 기반으로 하여 회원가입과 로그인을 진행하고
받고 싶은 선물에 대한 펀딩 글을 작성하거나 친구에게 선물해보세요 🎉🎉
```
⏰ 기간 : 2023.03.17 ~ 2023.06.20

📌 프로젝트 구분 : 교내 학업 프로젝트(모바일 앱 프로그래밍 2)

📃 내용 : 생일 선물 펀딩 어플

👥 참여자 : 김민영, 민시온, 정다혜, 문정원
